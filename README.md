# Auth0 (auth0)
Auth0 (now part of Okta) is a leading identity and access management platform providing secure authentication and authorization for applications, APIs, and devices. It implements OpenID Connect, OAuth 2.0, and SAML 2.0 protocols and offers a Management API, Authentication API, My Account API, and My Organization API for programmatic control over identity infrastructure.

## Features

| Name | Description |
|------|-------------|
| Universal Login | Centralized login page that can be customized and handles authentication flows for all application types. |
| Social Login | Integrate Google, Facebook, GitHub, Twitter, and 30+ social identity providers with minimal configuration. |
| Multi-Factor Authentication | Add TOTP, SMS, push notification, and WebAuthn MFA to any application using built-in Auth0 factors. |
| Single Sign-On | Enable SSO across multiple applications and APIs using standard protocols including SAML 2.0 and OpenID Connect. |
| Enterprise Federation | Connect enterprise identity providers including Active Directory, LDAP, Azure AD, and SAML IdPs for workforce authentication. |
| Actions | Customize authentication pipelines with serverless Actions that execute code at key points in authentication flows. |
| Organizations | Multi-tenant B2B identity management with per-organization branding, connections, and user management. |
| Anomaly Detection | Automatic protection against credential stuffing, brute force attacks, and suspicious login patterns. |

## Use Cases

| Name | Description |
|------|-------------|
| Customer Identity | Add secure, scalable authentication to customer-facing web and mobile applications with social login and passwordless options. |
| Workforce Identity | Federate with enterprise IdPs for employee authentication with SSO, MFA, and SCIM provisioning. |
| B2B Identity | Provide multi-tenant identity for SaaS applications with per-customer organization management and custom login flows. |
| API Authorization | Secure REST and GraphQL APIs using OAuth 2.0 access tokens with audience and scope validation. |
| Machine-to-Machine Auth | Issue OAuth 2.0 client credentials tokens for service-to-service API authentication without user involvement. |
